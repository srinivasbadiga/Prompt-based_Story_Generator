Project Overview

Prompt Story Generator

The Prompt Story Generator is an interactive storytelling system designed to create and evaluate stories using advanced prompting techniques like zero-shot, few-shot, and reflexion. Users can select a story title and genre or provide custom prompts, and the system generates tailored stories using a fine-tuned GPT model.

The generated outputs are evaluated using robust metrics such as ROUGE, BLEU, METEOR, and coherence to measure their quality and similarity to reference stories. The system features an intuitive web interface where users can visually compare these results through detailed graphs and confidence scores, gaining insights into the effectiveness of different prompting strategies.

This platform bridges creative writing and quantitative evaluation, making it a valuable tool for researchers and users exploring storytelling and prompt engineering. GitHub Repository: Prompt Story Generator

Prompting Techniques

Zero-shot learning means the model can perform a task without having seen any examples of it before. It relies on understanding from general knowledge.

Example: If you ask the model, "Translate 'Hello' into French," without giving any prior examples, it can still give the correct answer: "Bonjour."

Few-shot learning means the model can learn a task with only a few examples provided beforehand. It uses those examples to generalize and perform the task.

Example: If you provide 2-3 examples of translating English phrases to French, the model will use those to translate new phrases.

Reflexion refers to the model reviewing and correcting its own response by thinking about the task and refining its output.

Example: After generating an answer, the model might notice it missed a key detail and correct it: "Upon reflexion, I should also mention...".
